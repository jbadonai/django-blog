What the project does
---------------------

This is a simple django blog app which gives user opportunity to:
1. Register a new account
2. Login to post and edit post
3. Reset password
4. Ability to comment on other users post.


Why the project is useful
--------------------------
The project allows user to post what is trending for others to see.


How users can get started with the project
------------------------------------------
This project is currently being hosted on Heroku and can be accessed using the link below:
https://jbadonai-django-blog-app.herokuapp.com/

You can also download the repo from Git hub
